\section{\emph{simulated (latency 2)} latency model parameter deriviation} \label{sec:latencyparams}
We derived the values for the parameters, $l$ and $i$, for the \emph{simulated (latency 2)} from the dynamic model of \cite{Kim06}. Their model is as follows:
\[\frac{da}{dt} = -\omega(a-a_{\text{min}}) \]
\[\frac{dT^*}{dt} = (1-f)(1-\epsilon )kT_0V + aL - \delta T^*\]
\[\frac{dL}{dt} = f(1-\epsilon )kT_0V + rL - aL\]
\[\frac{dV}{dt} = N\delta T^* - cV\]
where $a$ is the activation rate of the latently-infected T-cells, $T^*$ is the number of actively-infected T-cells, $L$ is the number of latently-infected T-cells and $V$ is the numnber of HIV virions and $t$ is time given in days.
The parameters of the model are described in Table \ref{tab:modelparams}.
The values we used for the model parameters are also given in Table \ref{tab:modelparams} and are derived from their paper unless otherwise stated.

We took $l$ to equal their $a_0$, which is the initial value of the activation rate $a$.
In order for the model to give a realistic solution, \cite{Kim06} require that $a_0 \leq c\frac{V_0}{NL_0}-(1-f)(1-\epsilon)kT_0\frac{V_0}{L_0}$.
We follow them when they choose $a_0$ to be the maximum such value to get $l = a_0 = 5.8 \times 10^{-3}$ day$^{-1}$.
We took $i$ to be the infection rate of latently-infected cells: $f(1-\epsilon)k$.
Using this approach, we get $i = 2.0 \times 10^{-12}$ day$^{-1}$.
\end{comment}

\section * {Results with outgroup rooting} \label{sec:ogr}
Table \ref{tab:patientserrorogr} shows the results of the linear model on the data sets using outgroup rooting \cite{Huelsenbeck02} to root the phylogenies.
Five patients from the \emph{plasma} data set and seven patients from the \emph{mixed} data set were rejected by our screening process when using outgroup rooting.
Overall, outgroup rooting performed poorer than rooting with root-to-tip regression, in terms of the $p$-values of the linear models and the resulting root mean square distances (RMSDs).
The total scaled RMSD of the \emph{plasma} data set using outgroup rooting was $0.13$ and the total scaled median difference was $0.030$.
The total scaled RMSD of the \emph{mixed} data set using outgroup rooting was $0.52$ and the total scaled median difference was $-0.011$.
The binomial test on the \emph{plasma} data set predicted that there was no latency ($p = 0.7869$).
On the \emph{mixed} data set, the same test predicted that there was latency, but with marginal signifciance ($p = 0.0452$).
Also included in the Supplementary Materials are the corresponding figures of the real data from the primary work using outgroup rooting instead of root-to-tip regression (Figures S1-S4).
